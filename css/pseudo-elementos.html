Pseudos elementos

É usada para adcionara ou selecionar elementos do HTML sem precisar desclara-lo no documeto HTML

::first-letter

Seleciona a primeira letra da primeira linha e aceita as seguites propriedades, color, background, margin, paddin, border, font e propiedades de texto

::first-line

Seleciona a primeira linha do elemento selcionado, aceita as seguites propriedades, color, background, font e propiedades de texto

::backdrop

Ultilizado para mudar a cor de fundo, da area extra de uma tela de vídeo

::marker

Permite estilizar os topicos ou numeros de uma lista li, ou as flechas de um elemento sumary, aceita os seguites paramentros, color, content, white-space, font, animation, transition,

::placeholder

Permite estilizar o placeholder de um campo de formulario, aceita os seguites paramentros, color, font, background, text.

::selection

Permite estilizar a cor de um texto secionado pelo usuario, aceita os seguites paramentros, color, background-color, text.

::file-selector-button

Permite estilizar um botão de selecionar arquivos para fazer upload, aceita os seguites paramentros, color, background, text.

::before

Permite inserim umn elemento detro do elementro seliconado, antes de seu primeiro elemento filho, aceita os seguites paramentros, color, background, text.

::after

Permite inserim umn elemento detro do elementro seliconado, apos o ultimo elemento filho , aceita os seguites paramentros, color, background, text.
