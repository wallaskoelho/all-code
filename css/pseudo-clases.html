Pseudo Clases

:hover

Permite estilizar alguma elemento que sera mostrado quando o usuario pasar o mause por cima do elementor especificado

:active

Permite criar uma estilização de qualquer elemento equanto ele estive sendo clicdo

:focus

Permite criar estilização para quando um elemento estiver focado, selecionado, como um campo de um formulario quando for ciclado

:focus-within

Permiter apicar estilização para todos os elementos debtro da atributo label

:focus-visible

So aplica a estiliza quando o button for selecionado por movimento do teclado, e não aplica quando é clicado

:target

Permite estilizar um bloco do site quando ele for selecionado por meio de um link interno.

:enableb :disabled

Permite estilizar qualquer elemento que aceita a tag enabled e disabled, como a tag button por exemplo.

:link :visited

Permite estilizar um link, :link para links não clicados e visited para links visitados

placeholder-shown

Perminte estilizar um ipunt que tenha placeholder enquanto ele não estiver sido prenchido

checked

Perminte estilizar um um checkbox enquanto ele estiver selecionado

indeterminate

Perminte estilizar um elemento que posua uma condição de indeterminado

valid invalid

Perminte estilizar um campo de formulario quando houver espeficação de validade, e com uso de valid e invalid para demostra visualmente que o valor é valido ou invalido

in-range out-of-range

Perminte estilizar um campo de formulario, quando seu prechimento permite colocar um ranque de infromação, como idade, quantidade, atravez do atributo in-range e out-of-range desmotra visualmente se o valor inserio esta dentro ou fora dos valores aceitos como validos

required optional

Permiinte estilziar os campo de um formulario confore ele seja obrigatorio ou opcional

autofill

Permite estilziar um campo de um formulario que foi prenchido altomaticamente, desmotrando de forma visual ao usuario. e ideal usar o :-webkit-autofill para evitar incompatibilidades com alguns browser

root

Permite aplicar configuraçoẽs na raiz do site, o root equivale ao html.

nth-child nth-of-type

Permite eslizar os filhos de um terminado parmentro podendo ser especificado a ordem dos filhos, :nth-child(2)

nth-of-child nth-of-type

Permite eslizar os filhos de um terminado parmentro podendo ser especificado a ordem dos filhos, porem no sendtido do ultimo para o primeiro, de baixo para cima :nth-child(2)

O nth-child sempre vai estilizar o primeiro filho da mesma tag, e o nth-of-type vai selecionar todos so filhos independente do tipo de tag.

também pode selecionar uma lista de acordo como a posição se impar(odd) ou par(even)

first-fo-type last-of-type

Permite elstilizar o primeiro e ultimo elemento de cada tipo de tag

only-child

Permite estlizar intems que não tenha irmãos, independente do tipo de tag

only-of-type

Permite estlizar intems que não tenha irmãos de mesma tag, do mesmo tipo

mepty

Permiste selecionar elmentos que não tenha textos ecritos, elementos em branco, espaco, comentario, elemteos vasios dentro de elementos vasios, e texto inserio atravez do css são selecionados, são elementos vazios.

:is(){}

Permite selecionar varias tags, classes, id de uma so vez para adicionar qual paramentro css, :is(h2, p, .content1, #menu2){color:red} :is(main, article, div, section, span) p{color: blue; background: white

where

tem a mesma função que o :is, porem não tem a mesma formça, sendo rescrito caso o mesmo elemento seja estilizado por outra tag.

not()

É um elemento de engação, oque for declarado nele, não sera estilizado

:has()

Permite selecionar elementos com base nos elementos que ele posui
